# House Price Prediction using Machine Learning

This project predicts house prices using machine learning techniques. It includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation using Python and Scikit-learn.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Regression Model Training
- Model Evaluation
- House Price Prediction
